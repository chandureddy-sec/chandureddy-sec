A fully simulated **enterprise Active Directory environment** built from scratch to practice real-world domain penetration testing. | Attribute | Details |
|:---|:---|
| **Stack** | Windows Server (DC), Windows 10 Client, Kali Linux, BloodHound, Impacket, Mimikatz |

| **Security** | Intentionally misconfigured AD — weak service accounts, unconstrained delegation, ACL abuse |
| **Impact** | Full domain compromise demonstrated via privilege escalation from low-privileged user |
| **Repository** | [chandureddy-sec](https://github.com/chandureddy-sec) |

All findings were mapped to CVSS scores with documented remediation recommendations, mirroring a real enterprise pentest engagement deliverable.
